## Quick Start

1. Install Surfgram (CLI included):

    ```bash
    pip install surfgram
    ```

2. Create your first bot:

    ```bash
    surfgram new my_bot
    ```

3. Run the bot:

    ```bash
    surfgram run my_bot
    ```
